Exportar participantes

Los registros se guardan en Supabase. Para exportar los usuarios:

  1. Entra en tu proyecto de Supabase.
  2. Ve a Table Editor → tabla registrations.
  3. Filtra por toploader = true o keychain = true.
  4. Pulsa Export → CSV y copia la columna username.

O usa el SQL Editor con estas consultas:

-- Toploader
select username from registrations where toploader = true order by username;

-- Keychain / Llavero
select username from registrations where keychain = true order by username;
